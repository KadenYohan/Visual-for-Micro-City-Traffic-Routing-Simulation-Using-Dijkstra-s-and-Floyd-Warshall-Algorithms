Complexity Analysis

Dijkstra — O((V+E) log V)

Time: O((V+E) log V)  |  Space: O(V+E)

Best for single queries. Stops early when destination is found.

Floyd-Warshall — O(V³)

Time: O(V³)  |  Space: O(V²)

Best for many queries. Computes all V×V pairs in one pass.
